    Hardware

    My coumputer is a custom build system containing the following components:

    Component Name Description
    CPU AMD Ryzen 9 5950x The Amd Ryzen 9 5950x is a highend desktop processor which offers 16 native and 32 logical cores. Ther native frequency is 3.4GHz boosting up to 4.9-5GHz
    GPU Asus Rog Strix RX 5700 xt 8gb The Asus Rog Strix Rx 5700 xt 8gb is a very powerfull card, the 8gb of VRAM are enough to enjoy a high Frame Rate in most AAA games.
    RAM Gskill Trident Z Neo 4x 16gb The Gskill Trident Z NEO is a very powerfull ram the specs are talking for themself 3600MHz CL16 providing a butterly smoth expereience.
    Motherboard Asus X570-E Gaming All those powerfull components should be connected over a strong chipset, asus motherboad has big heatspreaders active cooling and a lot of other nice feautures, so it was the motherboard of my choice.
